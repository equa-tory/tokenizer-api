# tokenizer-api

`docker-compose up --build`

## OR

### Unix

cls; docker-compose down -v; docker-compose up --build

### Windows

docker-compose down -v && docker-compose up --build

## Scripts

### Unix

Then make it executable:

```bash
chmod +x start.sh
```

Run with:

```bash
./start.sh
```

### Windows

Run by double-clicking `start.bat`.
